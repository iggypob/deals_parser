#### Prerequisites (recommended)
* `rvm` (https://rvm.io/rvm/install)
* ruby 2.4+ (once you have rvm, run `rvm install 2.4`)
* `bundler` (https://bundler.io/)

#### How to run (for now tests only)
* Checkout the repository `git clone <repo_url>`
* run `bundle install`
* run `rspec`
